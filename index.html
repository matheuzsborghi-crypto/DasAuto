Passo a passo
Como configurar
Crie um token gratuito no GitHub. O Das Auto vai criar um Gist secreto chamado Das Auto Sync Database e guardar seus registros lá.
Crie ou entre na sua conta em github.com/signup. É gratuito.
Acesse github.com/settings/tokens.
Clique em Generate new token → Generate new token (classic).
Em Note, escreva algo como Das Auto App.
Em Expiration, escolha No expiration.
Em Select scopes, marque apenas gist. Nenhuma outra permissão é necessária.
Clique em Generate token e copie o token imediatamente. O GitHub mostra apenas uma vez.
Cole o token no campo acima e clique em Salvar. O Das Auto vai criar o Gist e sincronizar automaticamente.
